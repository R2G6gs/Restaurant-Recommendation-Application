▶️ 실행 방법
1️⃣ 저장소 클론
git clone https://github.com/R2G6gs/Restaurant-Recommendation-Application.git
cd Restaurant-Recommendation-Application

2️⃣ 가상환경 활성화 (Anaconda 기준)
conda activate base

3️⃣ 필수 라이브러리 설치
pip install -r requirements.txt

4️⃣ 프로그램 실행
python app_restaurant.py
